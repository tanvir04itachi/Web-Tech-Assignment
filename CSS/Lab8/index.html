CSS Overflow Properties

Understanding how content behaves when it exceeds container boundaries

What is CSS Overflow?

The CSS overflow property controls what happens to content that is too big to fit in an element's box. It's essential for creating clean, user-friendly layouts when dealing with dynamic content.

visible

Content renders outside the element's box (default)

hidden

Content is clipped and hidden from view

scroll

Scrollbars are always visible for navigation

auto

Browser decides when to show scrollbars

Overflow Property Examples

All examples below use a fixed container size of 300px × 200px with the same content.

overflow: visible

Default Behavior

Content extends beyond the container boundaries and remains fully visible.

Advanced Overflow Properties

CSS also provides specific properties for horizontal and vertical overflow control.

overflow-x & overflow-y

Control horizontal and vertical overflow independently.

Horizontal Scrolling Example

This content is intentionally very wide to demonstrate horizontal scrolling behavior when overflow-x is set to auto or scroll. You can see how the horizontal scrollbar appears when the content exceeds the container's width.

← Very Wide Content → ← That Requires Horizontal Scrolling → ← To View Completely → 
.x-scroll-box {
    overflow-x: auto;
    overflow-y: hidden;
    /* Horizontal scroll when needed */
    /* Vertical overflow hidden */
}

text-overflow: ellipsis

Handle single-line text overflow with ellipsis.

This is a very long text that will be truncated with an ellipsis at the end when it doesn't fit in the container width. 
.truncated-text {
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
    /* Single line text truncation */
    /* Shows "..." at the end */
}

Practical Use Cases

Chat Applications

Use overflow-y: auto for message containers to scroll through conversation history.

Data Tables

Apply overflow-x: auto to tables for horizontal scrolling on mobile devices.

Image Galleries

Use overflow: hidden for image containers with fixed aspect ratios.

Code Editors

Implement overflow: auto for code blocks with line numbers and syntax highlighting.
